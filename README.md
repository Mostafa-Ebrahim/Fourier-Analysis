# Fourier-Analysis
## Fourier analysis implementation codes for studying

### In this repository you will find:
- Fourier series implementation
- Gibbs phenomena visualization
- Using FFT to:
    - Denoise data
    - Compute Derivatives
    - Solve heat equation
- DFT matrix visualization
